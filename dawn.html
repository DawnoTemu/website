FAQ

Czy to naprawdę będzie brzmiało jak mój głos?

Tak! Używamy zaawansowanej technologii AI do klonowania Twojego głosu. Większość dzieci nie zauważa różnicy - dla nich to po prostu głos mamy czy taty opowiadający bajkę.

Ile czasu zajmuje nagranie próbki głosu?

Wystarczy około 3 minuty czytania przygotowanego tekstu. To jednorazowy proces - potem możesz korzystać z aplikacji bez ograniczeń.

Czy aplikacja zastępuje czytanie przez rodzica?

Absolutnie nie! DawnoTemu to narzędzie, które wspiera rodziców, gdy nie mogą być obecni lub potrzebują odpoczynku. Zachęcamy do regularnego, osobistego czytania dzieciom, gdy tylko jest to możliwe.

JavaScript Helpers

Dawn CSS zawiera pomocnicze funkcje JavaScript, które ułatwiają tworzenie interaktywnych elementów interfejsu.

Powiadomienia (Toasts)

Wyświetlaj użytkownikom nieinwazyjne powiadomienia za pomocą funkcji Dawn.notify().

Dawn.notify('Treść powiadomienia', 'success', 3000); // Typy: 'info', 'success', 'warning', 'error' // Ostatni parametr to czas w ms

Okna modalne

Twórz okna modalne dla ważnych komunikatów, formularzy i dialogów za pomocą funkcji Dawn.modal().

Dawn.modal( '<p>Treść okna modalnego</p>', { title: 'Tytuł okna', closeButton: true, width: '500px', onOpen: function() { console.log('Modal opened'); }, onClose: function() { console.log('Modal closed'); } } );

Akordeony

Komponenty akordeonowe przełączane za pomocą Dawn.toggleAccordion().

Sekcja 1: Podstawowe informacje

Akordeony są idealne do organizowania dużych ilości informacji w sposób, który nie przytłacza użytkownika.

Sekcja 2: Zaawansowane opcje

Możesz używać akordeonów do ukrywania zaawansowanych opcji, które nie są potrzebne większości użytkowników.

Sekcja 3: Dodatkowe informacje

W jednym momencie może być otwarty tylko jeden element akordeonu, co pomaga w skupieniu uwagi użytkownika.

<div class="dawn-accordion-item"> <div class="dawn-accordion-header">Tytuł</div> <div class="dawn-accordion-content"> <div class="dawn-accordion-body">Treść</div> </div> </div> // Przełączanie akordeonu element.addEventListener('click', function() { Dawn.toggleAccordion(this); });

Walidacja formularzy

Dawn JS zawiera wbudowaną walidację formularzy, która aktywuje się dla formularzy z klasą .dawn-form.

// Walidacja uruchamia się automatycznie // dla formularzy z klasą .dawn-form Dawn.initFormValidation(); // Sprawdza: // - Pola wymagane (required) // - Poprawność adresów email (type="email") // - Dodaje komunikaty błędów pod nieprawidłowymi polami
